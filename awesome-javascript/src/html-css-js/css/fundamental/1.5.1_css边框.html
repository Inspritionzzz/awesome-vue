border 属性

此属性是 border-width、border-style 以及 border-color 的简写属性。

border-left 属性

此属性是 border-left-width、border-left-style 以及 border-left-color 的简写属性。

border-bottom 属性

此属性是 border-bottom-width、border-bottom-style 以及 border-bottom-color 的简写属性。
